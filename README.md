# go-echo-basic-template
A basic REST API template for Go with echo

* Should provide following env variables
    * ENV
      * In Windows PowerShell
          ```bash
            $Env:ENV = 'application'
          ```
      * In Linux
          ```bash
            export ENV=application
          ```

    <br />

* Install dependencies
    ```bash
        go mod tidy
    ```


* Run application
    ```bash
        go run main.go
    ```
